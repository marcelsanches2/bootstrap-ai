# Skill Creator: Create New Project Kit

Crie um kit completo para uma nova tecnologia seguindo o padrão PaceBattle:

bootstrap → plan → jarvis-revisor → implementação → test-flow → ship

Regras:
- Use `plans/` para planos.
- Use `docs/ai/` como fonte de autoridade.
- Use `.claude/commands/product_roles/` para roles.
- Todo role precisa checklist e regra dura.
- Todo `test-flow` precisa comandos reais ou detecção explícita.
- Não sobrescreva arquivos de projeto destino silenciosamente.
